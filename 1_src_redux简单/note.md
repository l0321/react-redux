# redux

### 是什么
>redux是js的状态管理器。

适合哪些框架呢？ 
- react    用的最多的
- vue      vue中基本不用，有自己的vuex
- angular  很少用的

redux不是react插件，是一个独立的库

### 解决了什么问题

- 1：状态私有化，变为公共状态
- 2：非关系型组件的通信问题。
- 2：集中式状态管理
